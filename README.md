## What This Is

This repository contains an independent literature synthesis asking whether a hybrid mRNA vaccine framework could theoretically address key barriers in glioblastoma (GBM) immunotherapy. The three barriers examined are:

1. Weak immune activation in immunologically "cold" tumors
2. Blood-brain barrier (BBB) exclusion of therapeutic payloads
3. Immunosuppression within the tumor microenvironment

The proposed framework combines innate immune priming, patient-specific neoantigen targeting, BBB-penetrating delivery, and potential checkpoint inhibitor co-administration. All discussion is conceptual and grounded in published literature — no original experiments were conducted.

## Evidence Base

This work draws on published literature in:

- GBM immunotherapy and checkpoint inhibitor trials
- mRNA vaccine platforms
- Neoantigen identification and prediction methods
